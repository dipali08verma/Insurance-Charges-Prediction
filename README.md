# Insurance Charges Prediction

## Overview

This project uses Machine Learning techniques to predict insurance charges based on customer information such as age, BMI, number of children, smoking status, gender, and region.

## Dataset

The dataset contains the following features:

* Age
* Sex
* BMI
* Children
* Smoker
* Region
* Charges (Target Variable)

* Python
* Pandas

* Matplotlib
* Seaborn
